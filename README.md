# Daily Drinks Serveless Website Made by React

## System Requirement
* [Node.js](https://nodejs.org) v10.x
* [NPM](https://www.npmjs.com/) package manager
* [Yarn](https://yarnpkg.com) package manager

## Initialization
```
npm start
yarn start
```

## Useful Scripts

```
npm test
yarn test
```
Run test module

```
npm run build
yarn run build
```
Build production code
